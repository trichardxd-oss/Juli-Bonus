Conectar Firebase

Para que vos y Juli vean los mismos datos en tiempo real, necesitás conectar Firebase (gratis).

1. Andá a console.firebase.google.com
2. Creá un proyecto nuevo → dale un nombre
3. En el proyecto: Build → Realtime Database → Crear (elegí modo prueba)
4. En ⚙️ Configuración → Tus apps → Agregar app web → copiá los datos
5. Pegá abajo y tocá Conectar
